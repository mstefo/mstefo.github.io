Timely Repairs

What Is Timely Repairs?

Timely Repairs is a 2D puzzle / speedrun game where the player must repair as many clocks as possible within the 2 minute time limit. This game was submitted for the MACSJAM Semester 1 (2024). The game was developed to follow the theme "Analogue", with the player fixing analogue clocks.


Although this Game Jam was active for 48 hours, I was only a contributor for the second day
