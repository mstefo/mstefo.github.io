Untitled Pokémon Project

What Is Untitled Pokémon Project?

This project is an experiment to expand on my knowledge of the Unity engine. It is only made with the intention of testing the limits of what I can achieve and is not intended for any redistribution. This game is a 3D turn based battle game where teams of up to 6 Pokémon compete to complete specific objectives. The battle system takes strong inspiration from Pokémon Conquest (2012) and incorporates elements from Pokémon Masters EX (2019).

Key Contributions

As the sole contributor to this project, most features were developed by me. This includes the implementation of a complex turn based battle system, an "Event Queue" system, unique levels, fully functioning AI and all UI elements. Models, animations, sprites, icons and all audio were not created by me but imported through external tools such as Blender. Additionally, the model outline effect is done by Chris Nolet's Quick Outline package.

Battle System
  • Implemented most battle logic found in Pokémon Conquest
  • Ideated and added new features (e.g. weather, Sync Moves etc.) to the existing formula
  • Implemented an "Event Queue", which allows stat changes or modifications to apply sequentially
  • Multiple gamemode rulesets such as claiming all objectives, defeating the most AI controlled enemies etc.
  • Created and designed unique battlefields as well as rebuilt some from Pokémon Conquest
  • Incorporated fully functioning AI which properly weighs whether to priotise objectives or defeating opponents
