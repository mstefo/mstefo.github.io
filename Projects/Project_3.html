Brainybucks Financial Literacy Game (BFLG)

What Is BFLG?

BFLG is a 2D top down simulation game with the primary purpose of being an introduction to financial literacy concepts such as delayed gratification. The game involves the player recruiting and training inhabitants of Fishal Island to search for buried treasure in order to generate money. The player's goal is to reach a certain amount of money before 7 in game days pass. This project was developed during my time working as an intern for Brainybucks Consultancy.
