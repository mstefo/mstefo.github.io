Viva Madrid Mockup

What Is Viva Madrid Mockup?

Viva Madrid Mockup is, as the name implies, a demo website for the Giftwares store Viva Madrid. This was developed by request of the owner of the store in order to better solidify the brand and potentially increase exposure. The website is built using HTML and CSS, as the requirements involved simply showcasing the products in the store, their prices and other miscellaneous information. The mockup site can be accessed through the button below. Please note that some information is outdated or censored for privacy. This website is not compatible with mobile devices and may not render properly on smaller screens.
