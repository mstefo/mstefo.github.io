Memora

What Is Memora?

Memora is a photo storage and display application built using React Native. Optimised for Android device, it is intended to allow for the user to make collections of their photos and view them at any time. Users can also add extra information to each photo such as date taken, a description and categorise their photos for when they want to specifically view a certain type of photo.
